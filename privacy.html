HopDisk Privacy Policy

Last updated: June 21, 2026

Summary

HopDisk is a local macOS storage repair utility. It scans only folders or volumes you select, reads file metadata needed to calculate storage usage, and moves explicitly selected cleanup items to Trash. HopDisk does not upload scan results, file contents, file paths, or cleanup history to any server controlled by the app developer.

Data HopDisk reads

When you choose a folder or volume, HopDisk may read metadata inside that selected location, including:

  • File and folder names and paths
  • File sizes, allocated sizes, and package structure
  • Creation, modification, and access dates when available
  • Type identifiers, bundle identifiers, and owning app metadata when available
  • Whether an item appears to be a cache, log, package, virtual machine, app support file, or system-managed metadata

HopDisk uses this information to classify storage usage, detect storage mismatches, and build user-confirmed cleanup plans.

Data stored on your Mac

HopDisk may store scan history, item metadata, cleanup reports, saved scan locations, and security-scoped bookmarks locally on your Mac so you can review previous scans and rescan locations you chose to remember.

You can revoke saved locations in HopDisk settings. Removing the app or its local data removes locally stored HopDisk records.

Data collection and sharing

  • HopDisk does not use third-party analytics SDKs.
  • HopDisk does not track you across apps or websites.
  • HopDisk does not serve ads.
  • HopDisk does not sell or share user data.
  • HopDisk does not upload file contents, file paths, or scan results.

If you install HopDisk through the App Store, Apple may process purchase, download, crash, and diagnostic information according to Apple’s own policies and your device settings.

Permissions

HopDisk uses macOS user-selected file access. It scans only locations you select in the file picker or saved locations you explicitly choose to remember. HopDisk does not request root privileges, does not install privileged helpers, and does not run a background daemon.

Cleanup behavior

HopDisk moves selected cleanup items to Trash only. It does not permanently delete files and does not empty Trash automatically. You remain responsible for reviewing selected items and emptying Trash later if appropriate.

Support requests

If you contact support through GitHub Issues, information you post there may be public. Avoid sharing private file names, private file paths, screenshots, or logs unless you intentionally want to make them public.

Changes to this policy

This policy may be updated as HopDisk changes. The date at the top of this page shows when the policy was last revised.
