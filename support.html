English (en)

Support

Get help

For bug reports, questions, and support requests, use the public GitHub issue tracker: github.com/Facta-Leopard/HopDisk/issues. Include your macOS version, HopDisk version, scan type, and what you expected to happen.

GitHub Issues are public. Do not post private file paths, file names, screenshots, or logs unless you intentionally want to share them publicly.

Cleanup safety

  • HopDisk moves selected cleanup items to Trash only.
  • HopDisk does not empty Trash automatically.
  • HopDisk does not use root privileges, privileged helpers, or background daemons.
  • System-managed metadata, active files, virtual machines, and app data require review or are blocked.

System Data still looks large

macOS may count APFS snapshots, purgeable space, Spotlight indexes, package metadata, caches, and app support data as System Data. Some of those items are managed by macOS and are not safe cleanup targets. HopDisk separates safe candidates from items that require review.

Folder access

HopDisk scans only locations you select in the macOS file picker or saved locations you explicitly choose to remember. If items are inaccessible, reauthorize the saved location or select the relevant folder or volume again.

Privacy

HopDisk runs locally on your Mac and does not upload scan results. Read the Privacy Policy.

한국어 (ko)

지원

도움 받기

버그 신고, 질문, 지원 요청은 공개 GitHub 이슈에서 남겨 주세요: github.com/Facta-Leopard/HopDisk/issues. macOS 버전, HopDisk 버전, 실행한 스캔 종류, 기대한 동작과 실제 결과를 함께 적어 주시면 확인이 빠릅니다.

GitHub 이슈는 공개됩니다. 공개해도 되는 경우가 아니라면 개인 파일 경로, 파일명, 스크린샷, 로그를 올리지 마세요.

정리 안전성

  • HopDisk는 선택한 정리 항목을 휴지통으로만 이동합니다.
  • HopDisk는 휴지통을 자동으로 비우지 않습니다.
  • HopDisk는 루트 권한, 권한 상승 헬퍼, 백그라운드 데몬을 사용하지 않습니다.
  • 시스템 관리 메타데이터, 최근 사용 파일, 가상 머신, 앱 데이터는 검토가 필요하거나 정리 대상에서 차단됩니다.

시스템 데이터가 여전히 크게 보이는 경우

macOS는 APFS 스냅샷, 제거 가능 공간, Spotlight 인덱스, 패키지 메타데이터, 캐시, 앱 지원 데이터를 시스템 데이터로 계산할 수 있습니다. 이 중 일부는 macOS가 관리하는 항목이라 임의 삭제 대상이 아닙니다. HopDisk는 안전한 후보와 검토가 필요한 항목을 분리해서 보여줍니다.

폴더 접근

HopDisk는 macOS 파일 선택기에서 사용자가 직접 선택한 위치 또는 사용자가 저장하기로 한 위치만 스캔합니다. 접근할 수 없는 항목이 보이면 저장된 위치를 다시 승인하거나 해당 폴더 또는 볼륨을 다시 선택해 주세요.

개인정보

HopDisk는 사용자의 Mac에서 로컬로 실행되며 스캔 결과를 업로드하지 않습니다. 자세한 내용은 개인정보 처리방침을 확인해 주세요.

Français (fr)

Assistance

Obtenir de l'aide

Pour signaler un problème, poser une question ou demander de l'aide, utilisez le suivi public des issues GitHub : github.com/Facta-Leopard/HopDisk/issues. Indiquez votre version de macOS, la version de HopDisk, le type d'analyse lancé et le résultat attendu.

Les issues GitHub sont publiques. Ne publiez pas de chemins de fichiers privés, noms de fichiers, captures d'écran ou journaux si vous ne souhaitez pas les rendre publics.

Sécurité du nettoyage

  • HopDisk déplace uniquement les éléments sélectionnés vers la corbeille.
  • HopDisk ne vide pas automatiquement la corbeille.
  • HopDisk n'utilise pas les privilèges root, d'assistant privilégié ni de démon en arrière-plan.
  • Les métadonnées gérées par le système, fichiers actifs, machines virtuelles et données d'apps exigent une vérification ou sont bloqués.

Les données système semblent encore volumineuses

macOS peut compter les instantanés APFS, l'espace purgeable, les index Spotlight, les métadonnées de paquets, les caches et les données de support d'apps comme données système. Certains éléments sont gérés par macOS et ne sont pas des cibles de nettoyage sûres. HopDisk distingue les candidats sûrs des éléments à vérifier.

Accès aux dossiers

HopDisk analyse uniquement les emplacements que vous choisissez dans le sélecteur de fichiers macOS ou les emplacements enregistrés explicitement. Si des éléments sont inaccessibles, autorisez de nouveau l'emplacement enregistré ou choisissez à nouveau le dossier ou le volume concerné.

Confidentialité

HopDisk fonctionne localement sur votre Mac et n'envoie pas les résultats d'analyse. Consultez la politique de confidentialité.

Español (es)

Soporte

Obtener ayuda

Para informar errores, hacer preguntas o solicitar soporte, usa el rastreador público de incidencias de GitHub: github.com/Facta-Leopard/HopDisk/issues. Incluye tu versión de macOS, la versión de HopDisk, el tipo de análisis y qué esperabas que ocurriera.

Las incidencias de GitHub son públicas. No publiques rutas privadas, nombres de archivos, capturas de pantalla ni registros salvo que quieras compartirlos públicamente.

Seguridad de limpieza

  • HopDisk solo mueve los elementos seleccionados a la papelera.
  • HopDisk no vacía la papelera automáticamente.
  • HopDisk no usa privilegios root, asistentes privilegiados ni demonios en segundo plano.
  • Los metadatos gestionados por el sistema, archivos activos, máquinas virtuales y datos de apps requieren revisión o se bloquean.

Datos del sistema sigue pareciendo grande

macOS puede contar instantáneas APFS, espacio purgable, índices de Spotlight, metadatos de paquetes, cachés y datos de soporte de apps como Datos del sistema. Algunos elementos son gestionados por macOS y no son objetivos seguros de limpieza. HopDisk separa candidatos seguros de elementos que requieren revisión.

Acceso a carpetas

HopDisk analiza solo las ubicaciones que eliges en el selector de archivos de macOS o las ubicaciones guardadas que decides recordar. Si hay elementos inaccesibles, vuelve a autorizar la ubicación guardada o selecciona de nuevo la carpeta o el volumen correspondiente.

Privacidad

HopDisk se ejecuta localmente en tu Mac y no sube resultados de análisis. Lee la política de privacidad.

Deutsch (de)

Support

Hilfe erhalten

Für Fehlerberichte, Fragen und Supportanfragen nutze bitte den öffentlichen GitHub-Issue-Tracker: github.com/Facta-Leopard/HopDisk/issues. Gib macOS-Version, HopDisk-Version, Scan-Typ und das erwartete Verhalten an.

GitHub Issues sind öffentlich. Veröffentliche keine privaten Dateipfade, Dateinamen, Screenshots oder Protokolle, wenn du sie nicht bewusst öffentlich teilen möchtest.

Sicherheit bei der Bereinigung

  • HopDisk bewegt ausgewählte Bereinigungselemente nur in den Papierkorb.
  • HopDisk leert den Papierkorb nicht automatisch.
  • HopDisk nutzt keine Root-Rechte, privilegierten Hilfsprogramme oder Hintergrund-Daemons.
  • Systemverwaltete Metadaten, aktive Dateien, virtuelle Maschinen und App-Daten müssen geprüft werden oder werden blockiert.

Systemdaten wirken weiterhin groß

macOS kann APFS-Snapshots, löschbaren Speicher, Spotlight-Indizes, Paketmetadaten, Caches und App-Support-Daten als Systemdaten zählen. Einige dieser Elemente werden von macOS verwaltet und sind keine sicheren Bereinigungsziele. HopDisk trennt sichere Kandidaten von prüfpflichtigen Elementen.

Ordnerzugriff

HopDisk scannt nur Orte, die du im macOS-Dateidialog auswählst, oder gespeicherte Orte, die du ausdrücklich merken lässt. Wenn Elemente nicht zugänglich sind, autorisiere den gespeicherten Ort erneut oder wähle den betreffenden Ordner oder Datenträger erneut aus.

Datenschutz

HopDisk läuft lokal auf deinem Mac und lädt keine Scanergebnisse hoch. Lies die Datenschutzerklärung.

日本語 (ja)

サポート

ヘルプを受ける

不具合の報告、質問、サポート依頼は公開 GitHub Issues を使用してください: github.com/Facta-Leopard/HopDisk/issues. macOS のバージョン、HopDisk のバージョン、実行したスキャンの種類、期待した動作を記載してください。

GitHub Issues は公開されます。公開してよい場合を除き、個人のファイルパス、ファイル名、スクリーンショット、ログを投稿しないでください。

クリーンアップの安全性

  • HopDisk は選択したクリーンアップ項目をゴミ箱へ移動するだけです。
  • HopDisk はゴミ箱を自動的に空にしません。
  • HopDisk は root 権限、特権ヘルパー、バックグラウンドデーモンを使用しません。
  • システム管理メタデータ、使用中のファイル、仮想マシン、アプリデータは確認が必要、または対象外として扱われます。

システムデータがまだ大きく見える場合

macOS は APFS スナップショット、パージ可能領域、Spotlight インデックス、パッケージメタデータ、キャッシュ、アプリサポートデータをシステムデータとして数えることがあります。 その一部は macOS が管理しており、安全な削除対象ではありません。HopDisk は安全な候補と確認が必要な項目を分けて表示します。

フォルダアクセス

HopDisk は macOS のファイル選択画面で選んだ場所、または明示的に保存した場所だけをスキャンします。 アクセスできない項目がある場合は、保存済みの場所を再承認するか、対象のフォルダまたはボリュームをもう一度選択してください。

プライバシー

HopDisk は Mac 上でローカルに動作し、スキャン結果をアップロードしません。詳しくは プライバシーポリシー をご確認ください。
